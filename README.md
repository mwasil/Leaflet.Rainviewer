# Leaflet.Rainviewer

[Demo](https://mwasil.github.io/Leaflet.Rainviewer/demo/)

E.g.

````js
    // Change default options
    L.control.rainviwer({ 
        position: 'bottomleft',
        nextButtonText: '>',
        playStopButtonText: 'Play/Stop',
        prevButtonText: '<',
        positionSliderLabelText: "Hour:",
        opacitySliderLabelText: "Opacity:",
        animationInterval: 500,
        opacity: 0.5
    }).addTo(map);
````
